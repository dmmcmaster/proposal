Webware Final Project Proposal

Group Members: Kurt Bugbee, Derek McMaster, & Andrew Rottier


Project Idea:

     Our idea is to create a website destination for farmers' markets. We have data on almost 9000 farmers' markets across the country. We initially want to make this data come alive to the user by creating interactive data visualizations based on data characteristics such as average size, location, and annual duration. We also want to make it interactive by allowing users to look up locations to find markets near where they are or some place they will be. As of now we aren't sure of all of the ways we will present this data but we will for sure have multiple data visualizations using d3js or dimple and on a map using APIs similar to the Google Maps API.
